# Great tit genomics - Wytham Woods

This repository contains the computational pipeline used in **Estandia et al. (2025)** to explore dispersal and immigration in Wytham Woods, UK.  

---

## Citation  
If you use this pipeline or parts of it, please cite:  

**Estandia et al. (2025)**. *The genetic consequences of dispersal and immigration in a wild great tit population*. bioRxiv. 

---

## Data

The SNP datasets were published by Kim et al., 2018 Mol. Ecol. Resour. and van Bers et al., 2012 Mol. Ecol. Resour.
